_**Описание:**_

Репозиторий содержит код интернет-магазина с товарами из игры Animal Crossing: New Horizons. 

_**Функционал:**_

1. Просмотр каталога товаров: на сайте предоставлена информация о товарах (изображение, название, цена)
2. Работа с корзиной: пользователь может добавлять товары в корзину, удалять их и менять их количество
3. Оформление заказа: пользователь может открыть форму оформление заказа, где он вводит персональные данные (имя, фамилия, адрес доставки и контактный номер телефонв); после оформление заказа всплывает уведомление об успешном оформлении, и страница обновляется
4. localStorage сохраняет данные корзины на случай обновления страницы

_**Технологический стек:**_

Работа выполнена с использованием HTML, CSS, JavaScript.

_**Адаптивный дизайн:**_

Обеспечено корректное отображение сайта на разных устройствах (ноутбук, планшет и смартфон).
